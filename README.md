<h1>Cisco Packet Tracer Lab/Dynamic Routing/RIP</h1>



<h2>Description</h2>

In this basic lab I configure a virtual network using Cisco Packet Tracer. This will demonstrate how to configure and connect PCs, Switches ,and Routers from different networks using Dynamic Routing Information Protocol (RIP).
<br />


<h2>Utilities Used</h2>
 
- <b>Cisco Packet Tracer</b>

<h2>Environments Used </h2>

- <b>Windows 11</b>
